# AI-Powered Language Translation Tool

## Project Overview

This project is a multilingual language translation application built using Python, Streamlit, and Deep Translator. It allows users to translate text from one language to another through a simple and interactive web interface.

## Features

* Translate text between multiple languages
* Auto-detect source language
* User-friendly interface
* Real-time translation
* Error handling for empty input
* Supports multiple Indian and international languages

## Technologies Used

* Python
* Streamlit
* Deep Translator

## Installation

1. Clone the repository
2. Install required packages

pip install -r requirements.txt

3. Run the application

streamlit run app.py
